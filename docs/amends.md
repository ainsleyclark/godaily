# Amends

Ensure you read AGENTS.md and checkout to a new branch. Action the items below, if you are unsure,
ask.

- In ~/Desktop, there is footer-issue.png, you can see that the footer doesn't always sit on the
  bottom for both mobile and desktop viewports. No matter the size of the content, the footer should
  always float on the bottom.
- In ~/Desktop, there is button-issue.png, currently the share buttons wrap to the next line on
  mobile. It would be best to have the Share text, 100% width and the buttons underneath it. Ideally
  the buttons should sit nicely in that row by themselves but we need to be careful on smaller
  viewports.
- When I go to a single issue page, there's two dates. The first date has the current date, the
  black title has the date of the issue. We shoudl always display the date of the issue. You can
  also see this in button-issue.png
- In ~/Desktop, there is margin-issue.png. On mobile, I want to reduce the gap ever so slightly
  between the paragraph text in a section and the content.
- I want to remove the point score on digest items in the sample and email itself. For example, an
  item reading:xpzouying/xiaohongshu-mcp MCP for xiaohongshu.com xpzouying · 1 pts Should not have ·
  1 pts
- Sometimes if I click "Read on Reddit" for example in a Digest item, it goes to the link that the
  Reddit post points to. Not Reddit.com, we should always prioritise the original url in digest
  items.
